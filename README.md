# Sistema de Gerenciamento Jurídico - Desafio Técnico - Facilita Jurídico

Este projeto foi desenvolvido como parte do processo seletivo para a empresa Facilita Jurídico. O objetivo era criar um sistema de gerenciamento jurídico utilizando tecnologias modernas e práticas de desenvolvimento. O sistema permite o cadastro de clientes, advogados, processos e audiências, além de oferecer funcionalidades de busca e filtragem.

## Tecnologias Utilizadas

- **Node.js e Express**: Utilizados no desenvolvimento do backend da aplicação, proporcionando um servidor robusto e escalável para lidar com requisições HTTP.
- **React**: Utilizado na construção do frontend da aplicação, oferecendo uma experiência de usuário interativa e dinâmica.
- **Tailwind CSS**: Utilizado para estilização e design da interface, seguindo uma abordagem utility-first para uma estilização ágil e consistente.
- **Postgres**: Utilizado como banco de dados relacional para armazenamento e recuperação de dados relacionados aos clientes, advogados, processos e audiências.

## Funcionalidades Principais

- Cadastro de clientes, advogados, processos e audiências.
- Listagem e visualização detalhada de clientes, advogados, processos e audiências.
- Pesquisa e filtragem de clientes, advogados, processos e audiências.
- Interface amigável e responsiva para uma experiência de usuário agradável em dispositivos desktop e móveis.

## Pré-requisitos

- Node.js e npm instalados globalmente.
- Banco de dados Postgres configurado e acessível.

## Instalação

1. Clone o repositório: `git clone https://github.com/seu-usuario/nome-do-repositorio.git`
2. Navegue até o diretório do projeto: `cd nome-do-repositorio`
3. Instale as dependências do backend: `npm install`
4. Navegue até o diretório do frontend: `cd frontend`
5. Instale as dependências do frontend: `npm install`
6. Volte ao diretório raiz do projeto: `cd ..`
7. Execute o backend: `npm start`
8. Execute o frontend: `npm start`
